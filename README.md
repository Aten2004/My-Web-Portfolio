# Phurit (Aten) - Web Portfolio 2026

เว็บไซต์รวบรวมผลงานและทักษะส่วนตัว พัฒนาด้วย Next.js โดยเน้นดีไซน์เพื่อแสดงตัวตนในฐานะ Full-Stack Developer และ UX/UI Designer

## Overview
Portfolio นี้ถูกสร้างขึ้นเพื่อแนะนำตัวตน ประวัติการศึกษา ทักษะทางเทคนิค และโปรเจคต่างๆ ที่เคยทำมา โดยเน้นความลื่นไหลในการใช้งาน (UX) และความสวยงามแบบทันสมัย (UI) รองรับการแสดงผลทุกหน้าจอ (Responsive Design)

## Tech Stack
- **Frontend:** React, Next.js (App Router), TypeScript
- **Styling:** Tailwind CSS (v4)
- **Icons:** Lucide React, React Icons
- **Animation:** CSS Keyframes & Tailwind Utilities
- **Deployment:** Vercel

## Key Projects
- **Trading Journal:** ระบบบันทึกสถิติการเทรดทองคำแบบ Full-Stack
- **Any AI Caller:** การออกแบบ UX/UI สำหรับระบบ AI Agent Call Center
- **TU Lost & Found:** ระบบจัดการของหายที่เชื่อมต่อกับ AWS Backend (Lambda, DynamoDB)
